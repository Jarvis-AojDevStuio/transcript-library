## Context

Comprehensive UI/UX audit of the Transcript Library app followed by a ground-up redesign mockup. The app is a private desktop-first tool for friends to browse YouTube transcripts and curated analysis. Owner wants: no sidebar, cohesive single-space layout, large YouTube video, beautiful markdown analysis, full-width transcripts, visible topic metadata.

### Risks

- Mockup could look generic/AI-generated if not carefully designed
- Removing sidebar means navigation must be elegantly handled elsewhere
- Full-width transcript conflicts with readability (65-75ch line length)
- Video player sizing must balance with analysis content below
